    class DatastoreRepoOnAppScale

    An implementation of the Repository app that assumes it is running within AppScale. AppScale is assumed to start it up on its own.

    Constants

    NAME

    The name of this datastore, which we call AppDB since Neptune jobs basically use it as an interface to AppScale’s database agnostic layer.

    SERVER_PORT

    The port that the repo runs on, by default.

    Attributes

    host[RW]

    The host (IP, colon, port) that the Repo app is hosted at.

def initialize(credentials)
  repo_ip = "127.0.0.1"

  @host = "#{repo_ip}:#{SERVER_PORT}"
  HelperFunctions.sleep_until_port_is_open(repo_ip, SERVER_PORT,
    HelperFunctions::DONT_USE_SSL)
end
